Bonjour ce site a été créé dans le cadre d'un projet pour notre premier semestre en IUT informatique. Pour ce projet nous avons dû interviewer une personne qui travaille dans l'informatique ou bien qui utilise l'informatique au quotidien. Nous avons choisi d'interviewer le médecin Sarah Lagrogerie, et ce site vous montrera les différentes démarches et productions que nous avons réalisées.
